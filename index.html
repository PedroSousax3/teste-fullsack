Dependências

Pagina index

Na pagina index.js, que é criada quando criamos nossa aplicação react, nela temos uma função muito importante que é a transformação de todas nossa paginas e componentes em um unico elemento no html, que é o elemento que leva o id root.

Para que isso funcione primeiro temos duas impotações importantes:



import React from 'react';
import ReactDOM from 'react-dom';

Com esse dois elementos já importados devemos entã fazer uso da função render do componente ReactDOM.







ReactDOM.render(
  <React.StrictMode>
    //Codigo
  </React.StrictMode>,
  document.getElementById('root')
);

Como pode ser visto no modelo a função render recebe dois parametros o primeiro seria o React.StrictMode que teria os elementos do react a serem usados, e o segundo o elemento que a receberá.

Configurando Routes

Para fazermos bom uso do react devemos utilizar as Single Page Application (SPA) na criação de nosso site para isso utilizamos um framework chamada de React-Router-Dom.

  1. Instalar dependencia
    npm install react-router-dom
  1. Na pagina router.js importe o BrowserRouter, Switch, Route
    import { BrowserRouter, Switch, Route } from 'react-router-dom';

  1. Neste componente que ira criar deve ter em mente que ele deve retornar um BrowserRouter que dentro dele terá um Switch.

  2. Neste passo deve-se fazer 2 coisa, primeiro importa a pagina e depois





















import React from 'react';
import { BrowserRouter, Switch, Route } from 'react-router-dom';

import Home from './pages/Home/index.js';
import Inserir from './pages/Inserir/index.js';
import Alterar from './pages/Alterar/index.js';

export default function Rotas(){
    return (
        <BrowserRouter>
            <Switch>    
                <Route path = "/" exact = {true} component = {Home} />
                //path é ultilizado para dizer qual será a extensão da url desta pagina
                <Route path = "/Cadastrar" component = {Inserir} />
                <Route path = "/Alterar" component = {Alterar} />
            </Switch>
        </BrowserRouter>
    );
}
  1. No ultimo seria a ultilização do componente Link no lugar do a(no Link to = href porem por ele a pagina não recarregara e sim rederizar novamente).






import { Link } from 'react-router-dom';//importação do componet Link do react-router-dom
<Link to = "/">
//to indicando que a url deve ser /
    <img src = {logo} alt = "BLACKLIST" />
</Link>

Usando as rotas diretamente no o arquivo index.js do src

Para isso basta que no componente index.js importemos o componente router.js e colocalo dentro da função render no componente React.StrictMode.
HTML
















<!DOCTYPE html>
<html lang="pt-br">
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    
    <link rel="stylesheet" href="./reset.css">

    <title>Teste React</title>
  </head>
  <body>
    //div com o id root onde será renderizado todos os components e criados na pasta src 
    <div id="root"></div>
  </body>
</html>

JavaScript












import React from 'react';
import ReactDOM from 'react-dom';

import Rotas from './Router.js';

ReactDOM.render(
  <React.StrictMode>
    <Rotas />
  </React.StrictMode>,
  document.getElementById('root')
);

Pasta Services

Esta por si já é uma pagina muito importante pois ele que tem a responsabilidade de se conectar o frontend e a api.

Para seralizar essa conecção precisamos do link dessa api.

  • Instalar dependencia Axios
    npm install axios
  • Codigo demostrativo do uso do exios:





















//importação da dependencia axios para axiliar na coneção com a api
import axios from 'axios';

const api = axios.create({
    //Url onde está localizada a api
    baseURL: 'http://localhost:5000'
});

export default class ListaNegraApi {

    //async e await são usado pois não temos o resultado de aip no mesmo momento então usamos o async await por esse motivo
    async consultar(){
        //verbo ultilizado nesta fução é o get, passando o parametro o restante do caminho até a api para encontrar esta função que ira realizar a consulta.
        const result = await api.get('/lista');
        
        //retorna um modelo com diversos valores, o que queremos nesta explicação é a data que são os dados que a aip volta do banco de dados.
        return result.data;
    }
}
